## Navigation Button

These should only be used for navigation elements that require buttons and never for actions. They should not be used for "next" or "previous" navigation in forms.

{(() => {
const arrowRight = React.createElement(ArrowRight);
return (

<div className="flex gap-2">
  <Button variant="navigation" size="lg" suffix={arrowRight}>
    Navigation LG
  </Button>
  <Button variant="navigation" size="md" suffix={arrowRight}>
    Navigation MD
  </Button>
  <Button variant="navigation" size="sm" suffix={arrowRight}>
    Navigation SM
  </Button>
</div>
); })()}
